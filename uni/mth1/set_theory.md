### Power set $\mathcal{P}(X)$ 

<details>
<summary class="button">
Example
</summary>

$$
X = {1,3}\\
\mathcal{P}(X) = \{\{\emptyset\}, \{1\}, \{3\}, \{1,3\}\}
$$
</details>

The power set contains $2^n$ sets where n is the number of elements in the set
